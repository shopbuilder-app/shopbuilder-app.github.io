Build App

After customize the Design and configure the Features of your App. Now let's build the app.

Table of content:

1. Build Android app​

On the left menu, select Build > Play Store > I have an account.

There are 4 steps. This guide will focus to step 3:

  • Enter your email and click APPLY to receive the result of the build.
  • Select App Bundle or APK
  • [UPDATE BUNDLE ID] Enter your Bundle ID and click UPDATE (this guide to create the app Bundle ID). Check the Use OneSignal checkbox if you would like to push notifications via OneSignal.
  • [KEYSTORE FOR ANDROID] 💠 Enter your Keystore's information if you already had them. (this guide to create Keystore manually) 💠 Or you can create Keystore automatically by selecting CREATE KEYSTORE button, then:  1. input a string to the key password and store password _textboxes (they must be the same, at least 6 characters) 
  1. select_ CREATE button. 

💠 Input your Google services file by select the folder icon (search "2. Android" in this guide to download your google-services.json__)

Then click Submit Build button. The result and APK, AAB files will be sent to your email after 15-30 minutes.

Finish!

2. Build iOS app​

On the left menu, select Build > App Store > I have an account.

There are 2 steps. This guide will focus to step 2:

  • Enter your email and click APPLY to receive the result of the build.

  • [UPDATE BUNDLE ID] Enter your Bundle ID and click UPDATE (this guide to create the app Bundle ID). Check the Use OneSignal checkbox if you would like to push notifications via OneSignal.

  • [CERTIFICATE FOR iOS] Input information of Apple Store Connect, Certificate, Provisioning, Google service files as below: 💠 Enter information of your Apple Store Connect (please create an app on AppStoreConnect with Admin access). 💠 Input your certificate file .p12 and its password. If you do not have it, select Automatically to create a new one. 💠 Input your Provisioning file or click Create provisioning button to create it automatically. (To create Provisioning manually, please follow this 

    Make sure to select Re-create if exists to delete the old provisioning file. 💠 Input the GoogleService-Info.plist file by selecting the folder icon (search "1. iOS" in this guide to download your GoogleService-Info.plist file)

    > If getting an issue when creating a certificate, because **it already exceeded the number of certificates** with Distribution and iOS Distribution permissions, please go to [Apple Developer ](https://developer.apple.com/)to Revoke one as these steps:       
    >1. [Login](https://tppr.me/76rMT) with your dev account       
    >2. Click [Certificates, Identifiers & Profiles](https://tppr.me/FS5nt)       
    >3. Select an [old Distribution or iOS Distribution certificate](https://tppr.me/QJfko)        
    >4. Select [Revoke](https://tppr.me/2Dn2N) - now you can continue to `Automatically` to create a new certificate.

  • [BUILD ON CLOUD] 💠 Input app information as below: App name (at least 2 characters) App icon App version (format of x.x.x)Build version: 💠 Remember increasing this number of build versions whenever you build.

💠 Please enable Export env.dart & config JSON automatically option to build your app with your current configuration on the tool. Or if you want to upload the configuration from your local, refer to the env.dart and config_en.json files.

💠 Select Upload information button.

Then click Submit Build button. If building the IPA file successfully, it will be uploaded to TestFlight. The result will be sent to your email after 1-2 hours (then wait 30 minutes for TestFlight approval).

You can also click Integration Configs button to see if all the required configurations are done or not.

Or click Download to download all information of your app, e.g. Certificate, Provisioning files…

Finish!

3. View your iOS app​

After receiving the email that notifies you have built the iOS app successfully, follow the steps below to test your iOS app on your smartphone:

  • Go to App Store Connect, then sign in with your Apple ID.

  • Select My Apps

  • Select your app, then choose TestFlight

  • Now you can check the App Version and the Build Version to view the correct one have built:

  • Open TestFlight on your iOS smartphone to test the app. Enjoy!
